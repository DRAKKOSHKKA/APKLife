# Contributor Covenant Code of Conduct / Кодекс поведения сообщества

## Our Pledge / Наши обязательства
We as members, contributors, and leaders pledge to make participation in our community a harassment-free experience for everyone.
Мы как участники, контрибьюторы и мейнтейнеры обязуемся поддерживать безопасную и уважительную среду для всех.

## Our Standards / Наши стандарты
Examples of behavior that contributes to a positive environment / Примеры позитивного поведения:
- Being respectful and empathetic / Уважительное и эмпатичное общение.
- Giving and accepting constructive feedback / Конструктивная обратная связь.
- Taking responsibility and learning from mistakes / Ответственность и готовность учиться на ошибках.

Examples of unacceptable behavior / Примеры неприемлемого поведения:
- Harassment, trolling, and personal attacks / Травля, троллинг и персональные оскорбления.
- Publishing others' private information / Публикация чужих персональных данных.
- Any conduct inappropriate in a professional setting / Непрофессиональное поведение.

## Enforcement Responsibilities / Ответственность за соблюдение
Project maintainers are responsible for clarifying and enforcing acceptable behavior.
Мейнтейнеры проекта отвечают за разъяснение и применение правил поведения.

## Scope / Область действия
This Code of Conduct applies within all project spaces and public representation.
Кодекс действует во всех пространствах проекта и в публичном представлении сообщества.

## Enforcement / Применение мер
Report incidents to project maintainers through repository issues or direct contact.
О нарушениях сообщайте мейнтейнерам через issues репозитория или напрямую.
