# 📘 APK Life

**APK Life** — веб-приложение для студентов и преподавателей **АПК Альметьевск**.
Оно адаптировано под смартфоны, планшеты и ПК, а также поддерживает работу при временной недоступности источника расписания.

## ✨ Возможности

- 📅 Просмотр расписания по группе, преподавателю или аудитории.
- 🔍 Быстрый поиск с подсказками.
- 🌗 Тёмная тема с локальным сохранением настройки.
- 💾 Локальное кэширование расписания в файл (серверная часть) и автоматический fallback при проблемах с сетью.
- ⏱ Если внешний сайт не отвечает дольше 30 секунд, используется последняя сохранённая версия расписания.
- 🔄 Проверка актуальности локальной версии относительно GitHub-репозитория.

## 🧩 Технологии

- **Backend:** Python 3, Flask, Requests, BeautifulSoup4
- **Frontend:** HTML5, CSS3, Bootstrap 5, Bootstrap Icons, JavaScript
- **Data:** JSON (подсказки + локальный файловый кэш расписания)

## 🚀 Локальный запуск

```bash
pip install flask requests beautifulsoup4
python app.py
```

После запуска приложение будет доступно на `http://127.0.0.1:5000`.
