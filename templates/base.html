О приложении APKLife
APKLife помогает студентам и преподавателям быстро смотреть актуальное расписание с телефона, планшета и ПК. При проблемах с интернетом приложение автоматически попытается открыть локально сохранённую версию расписания.
Технологии:
  • Flask + Python (сервер и парсинг данных)
  • HTML5, CSS3, Bootstrap 5, Bootstrap Icons
  • JavaScript (тёмная тема, подсказки, localStorage)
  • Локальный кэш расписания (offline fallback)
